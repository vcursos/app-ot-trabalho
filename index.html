❓ Guia Rápido do Aplicativo

1. Registrar OT: Preencha número (opcional), selecione serviço e adicionais, adicione equipamentos (pode escanear), clique em Registrar OT.

2. Registrar Logística: Na aba Logística, a data de hoje já aparece. Preencha KM inicial, ao final do dia KM final, abastecimento (se houve) e clique em Registrar Jornada.

3. Resumos: Cartões no topo mostram quantidade e valor de hoje e do mês.

4. Filtros & Busca: Use mês, categoria, rede ou pesquise por código de barras para localizar rapidamente.

5. PDF Mensal: Selecione o mês e clique em Gerar PDF (OT ou Logística) para criar relatório offline. O arquivo é salvo na pasta de downloads do dispositivo.

6. Backup JSON: Botão Backup JSON gera um arquivo contendo TODOS os dados (OT + Logística). Faça ao fim do mês antes de limpar ou trocar de aparelho.

7. Importar Backup: Botão Importar Backup permite escolher um arquivo JSON exportado anteriormente. Você decide se substitui tudo ou se mescla sem duplicar.

8. Limpar Dados: Abra limpar-dados.html para remover completamente o histórico local (irreversível). Faça somente após garantir o backup.

9. Segurança: Os dados ficam apenas no dispositivo. Se desinstalar ou limpar cache, sem backup você perde o histórico.

Fluxo mensal sugerido: Registrar diariamente → No último dia do mês gerar PDF OT + PDF Logística → Fazer Backup JSON → (Opcional) Limpar para iniciar novo mês mais leve.
